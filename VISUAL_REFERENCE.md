   ├─ Feature Image (Visual anchor)

## User Flow

```
1. User scrolls down from Hero section
2. Core Features title comes into view
3. First feature block fades in and slides up
4. User continues scrolling
5. Each subsequent block animates as it enters viewport
6. User reads feature details and views illustrations
7. User understands Navin's core capabilities
8. User continues to next section (Features grid, How It Works, etc.)
```
